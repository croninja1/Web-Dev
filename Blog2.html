Week 2 Blog

In week 2 we focused much more on the CSS portion of code which helps more with the
visual aspect of the webpage. CSS is important because you use it to make a website more
appealing rather than just a white screen with black text. I decided to research more into CSS
and what all can be done with it. I started by looking at a w3schools page that showed the
different ways to style html using CSS. I first learned that there are 3 different ways include CSS
to html. The first is inline where you use a style attribute such as

<h1 style =”color: green;”> This heading is green</h1> The second is Internal where you add a tyle element in the head
attribute at the top, and the third is external where you create a CSS file and then link that within the
html file. All of these are good options depending on what you are trying to do.

After going over the basics such as changing the background/text color, changing the
font, and adding borders, I wanted to go more in depth and learn about some more complex
topics. I read an article about web style sheets and learned many new selectors and CSS
code styles that could come in handy. One that was very interesting was the way you could
style images and other figures with captions. You would do this using the figure tag in html
and then using CSS can position where the text is, how big the image is, the margins of the
image and the padding. An example of the CSS code for a figure would look something like this:

Figure { Width: 30%; Text-align: center; Border: solid; Margin: 0.5em; Padding: 0.5em; }

This would have the image filling almost the entire box with the caption centered perfectly
under it. This would be very helpful when making a website because every good website has a
good use of images. When you have your images formatted well, your website looks much more visually
appealing and more professional.
